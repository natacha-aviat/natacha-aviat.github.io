## üõ†Ô∏è Tools Developed for MaMaMa Association

The [MaMaMa association](https://www.mamama.fr) fights against child malnutrition in √éle-de-France. I have developed several operational tools to facilitate their field actions.

### [MaMaExpress](MaMaExpress.html)
**Logistics Route Optimization Tool**

Web application to optimize delivery routes to PMI (Maternal and Child Protection Centers):
- Automatic calculation of optimized routes
- Cartographic visualization of routes
- Management of multiple addresses and trucks
- Export to Google Maps

**Technologies** : JavaScript, Leaflet, optimization algorithms

### [MaMaFoodScan](MaMaFoodScan.html)
**Food Product Scanner**

Mobile-friendly application to scan and analyze food products:
- Barcode scanning (QuaggaJS)
- Nutritional data retrieval (Open Food Facts)
- Nutri-Score display
- Data export to Google Sheets
- Product list management

**Technologies** : JavaScript, QuaggaJS, Open Food Facts API

### [SaveEat](SaveEat.html)
**Recipe Manager**

Web application to collect and organize recipes:
- Manual recipe addition
- URL reception (iPhone Shortcuts compatible)
- Status management (To Cook, Validated, Rejected)
- Data Export/Import
- Local storage (localStorage)

**Technologies** : Vanilla JavaScript, localStorage
