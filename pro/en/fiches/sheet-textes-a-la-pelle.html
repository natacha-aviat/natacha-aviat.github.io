Textes à la pelle

Web page aggregating French writing contests, short story competitions and calls for submissions. Announcements are synced daily from textes-a-la-pelle.fr, with a filtering and sorting interface to quickly spot relevant opportunities.

❌ Before

  • Announcements scattered across multiple sites
  • Hard to filter by genre, fees or target audience
  • No consolidated up-to-date view

✅ After

  • Single list, updated daily
  • Filters by type, genre, entry fees and audience
  • Sort by publication or closing date
  • Live results counter

How it works

1. Data sync:

A GitHub Actions workflow regularly fetches announcements from textes-a-la-pelle.fr and stores them in a local JSON file.

2. Interactive interface:

Using JavaScript, the page loads announcements, generates filters dynamically and displays sorted results without reload.
