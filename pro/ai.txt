# Description du service
Natacha Aviat propose des services de cartographie interactive et d'analyse de données pour managers techniques, associations, entreprises engagées et institutions publiques. L'objectif est de fournir des insights actionnables pour les équipes, permettant de visualiser les données sur une carte et de prendre des décisions éclairées.

# Services proposés
1. Cartographie interactive
   - Création de cartes interactives avec Leaflet
   - Visualisation de données géographiques
   - Cartes sur étagère (0-600€) ou sur-mesure (1500€+)

2. Analyse de données
   - Exploration et nettoyage de données
   - Visualisation de données
   - Aide à la décision basée sur les données
   - Extraction d'insights actionnables pour les équipes

3. Développement d'outils opérationnels
   - Outils web sur mesure
   - Optimisation de processus
   - Automatisation de tâches

# Public cible
- Managers techniques (chimie, BTP, environnement, dépollution)
- Associations et ONG
- Entreprises engagées
- Institutions publiques

# Technologies utilisées
