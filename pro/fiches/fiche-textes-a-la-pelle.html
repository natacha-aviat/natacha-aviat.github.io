Textes à la pelle

Page web qui agrège les concours d'écriture, concours de nouvelles et appels à textes francophones. Les annonces sont synchronisées quotidiennement depuis textes-a-la-pelle.fr, avec une interface de filtrage et de tri pensée pour repérer rapidement les opportunités pertinentes.

❌ Avant

  • Annonces éparpillées sur plusieurs sites
  • Difficile de filtrer par genre, frais ou public cible
  • Pas de vue consolidée à jour

✅ Après

  • Liste unique, mise à jour quotidiennement
  • Filtres par type, genre, frais d'inscription et public
  • Tri par date d'ajout ou date d'échéance
  • Compteur de résultats en temps réel

Fonctionnement

1. Synchronisation des données :

Un workflow GitHub Actions récupère régulièrement les annonces depuis textes-a-la-pelle.fr et les stocke dans un fichier JSON local.

2. Interface interactive :

En JavaScript, la page charge les annonces, génère les filtres dynamiquement et affiche les résultats triés sans rechargement.
